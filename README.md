# Dj01
Простое приложение Django, в котором при переходе на страницу data и test открываются страницы с разным содержимым:
- `/data/` - страница с данными,
- `/test/` - тестовая страница,
- а на каждой из этих страниц дополнительно есть ссылки для перехода между страницами (на соседнюю страницу — одну из этих двух);
- откройте в браузере:
1. -> http://127.0.0.1:8000/data/ ←
2. -> http://127.0.0.1:8000/test/ ← 
3. -> Переход по ссылкам между страницами ← 
## Технологии
- Python 3.8+
- Django 6.0+
## Структура проекта
- `WebDjProject/` - основные настройки Django
- `myapp/` - приложение с двумя страницами
- `myapp/views.py` - функции для страниц data и test
- `myapp/urls.py` - маршруты приложения
